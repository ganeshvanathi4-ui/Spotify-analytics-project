# Spotify Analytics & Recommendation System 🎧

## 📌 Project Overview
This project analyzes Spotify listening data and builds a recommendation system.

## 🛠 Tech Stack
- Python (Pandas, Scikit-learn, Spotipy)
- SQL
- Power BI
- Airflow

## 📂 Project Structure
- data/ → raw & processed datasets
- scripts/ → ETL pipeline
- ml/ → recommendation system
- dashboard/ → Power BI dashboard
- airflow/ → workflow automation"# Spotify-analytics-project"
